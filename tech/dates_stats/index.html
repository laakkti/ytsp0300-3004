Getting NDVI dates and statistics of selected AOI

sequenceDiagram autonumber participant Frontend participant Backend Frontend->>Backend: POST /ndvi/api/v1/list (params in body,TOKEN in header) activate Backend alt success Backend->>Frontend: status 200 (json array of data) else not success Backend->>Frontend: status 404 (not found) deactivate Backend end
  1. request for list of NDVI datas & statistics, needs token in header, token has been given in result of succesfull authentication parameters geometry of AOI, startDate and endDate as body of message
  2. returns arrays of data with statuscode 200
  3. returns error status code 404
