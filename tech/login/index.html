sequenceDiagram autonumber participant Client participant Server participant MongoDb Client->>Server: POST /ndvi/api/v1/login {username, password} alt Authentication Successful Server->>MongoDb: Validate user credentials MongoDb-->>Server: Acknowledge validation Server-->>Client: Status 200 (Login successful, TOKEN returned as message body) else Authentication Failed Server-->>Client: Status 401 (Login failed, invalid credentials) end
  1. request for login to backend route with usernama and password
  2. validate credentials
  3. returns information of the success or not
  4. if insertion is successful status code 200 is returned with TOKEN as message body
  5. if insrtion fails status code 401 is returned
