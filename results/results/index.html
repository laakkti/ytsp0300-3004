Best practices

I have kept in mind reusability, maintainability and further developability.

TypeScript

It would be not bad idea to use Typescript instead of Javascript

Redux or Context for global variables

For state management would be handy in some deep hierarcy to use global approach.

Local storage/ Token

Token could be stored in the local storage and give expiration time. Local Storage is a key-value database in the browser. Values in the local storage are persisted even when the page is re-rendered.

JavaScript Standard Style

https://standardjs.com//

Naming policy

Should use better some naming policy for variables, properties, functions, methods, components etc.
